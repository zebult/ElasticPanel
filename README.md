## ElasticPanel

this app is create elastic panel.

### elastic panel?

elastic panel is change panel size for scroll and key input.

### when use?

read e-book etc.

### usage

- scroll: change panel size(up, down)
- key j : change panel size(down)
- key k : change panel size(up)
- key r : reset panel size
